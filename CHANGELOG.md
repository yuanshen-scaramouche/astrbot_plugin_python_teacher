## [1.0.0] - 2026-05-10 🎉

### 正式版发布！

- 系统化 Python 学习路线（6 个模块，40+ 课）
- 支持自定义 AI 接口配置（OpenAI 兼容）
- 支持多文件上传和引用
- 支持 AI 对话上下文记忆（10 轮对话）
- 支持课程切换（/py 下 / /py 上）
- 支持清空对话记忆（/py 清空记忆）
- 支持练习、答案提交、AI 点评、小测
- 作者：星汐

## [0.8.0] - 2026-05-10

- 新增 `/py 清空记忆` 指令，用于清空 AI 的对话历史

## [0.6.0] - 2026-05-10

### Added
- 大幅扩展文件格式支持：常见代码格式（py/js/ts/c/cpp/go/java/php/rb/rs）、配置文件（toml/json/yaml/ini）、文档（md/txt/rst）、Web（html/css/vue/svelte）等都已支持上传。其他文件格式暂不支持，可以联系作者或者自行添加。（意义为防止用户上传一些无法识别的文件如文件（如：.exe、.dll、.sys 等），方便AI解答）

## [0.5.0] - 2026-05-10

### Added
- `/py 问 <文件数量> <问题>`：支持上传文件并让 AI 看到文件内容
- 如果只指定文件数量而没有问题，会提示补充问题

## [0.4.0] - 2026-05-10

### Added
- `/py 下` / `/py next`：快速进入下一课
- `/py 上` / `/py prev`：快速进入上一课
- 每课页底提示切换课的指令

## [0.3.0] - 2026-05-10

### Added
- 支持上传多个文件，通过 `/py 答案 <数量>` 指定使用最后 n 个文件（1-5）
- 每个文件会标注文件名，让 AI 清楚知道这是多个文件

## [0.2.0] - 2026-05-10

### Added
- 支持上传文件作为答案：`.py` / `.txt` / `.md` / `.ipynb` / `.json` / `.yaml` / `.csv` 等
- 自动检测文件编码

## [0.1.0] - 2026-05-10

### Added
- 初始版本：Python 学习插件（路线、课程、练习、测验、AI 答疑）
